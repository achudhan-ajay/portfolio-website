Page Loader V1 – Counter (HTML/CSS/JS + GSAP)
• A premium full-screen page loader featuring a synchronized percentage counter and progress bar. Built for smooth transitions and modern landing experiences.
● Features
1. GSAP-powered smooth animations
2. 0 → 100 animated percentage counter
3. Synced bottom progress bar
4. Clean slide-up reveal transition
5. Fully responsive
● How It Works
1. #loader covers the viewport.
2. GSAP animates a value from 0 → 100.
3. onUpdate syncs: 
• Counter text
• Progress bar width
4. On completion, the loader slides out to reveal content.
